# 命名规范

## 一.工程名

> 规则：公司缩写 - 项目含义

## 二.包名

- 应用

>规则：com.公司名.工程名小写

- 系统应用

- 应用类分包

	- adapter:适配器

	- common:通用

	- utils:工具类

	- manager:管理类

	- impl:接口

	- http:网络相关

	- base:基类

	- fragment:碎片

	- activity:活动

	- view：自定义

	- model:数据模型

	- 待补充

>规则：com.android.工程名小写

## 三.类名

- Activity

	- 功能 + Activity,比如设置：SettingActivity

- Fragment

	- 功能 + Fragment,比如设置：SettingFragment

- 工具类

	- 功能 + Utils,比如日志：LogUtils

## 四.常量名

- Handler What

	- H_xx_xx_xx 比如系统时间：H_SYSTEM_TIME = 0x11;
